# Musify Study project

An ASP.NET Core Web API project, aimed at exploring the features of the platform.

# Overview

This project aims to implement an ASP.NET Core Web API that should serve a webshop revolving around musical instruments. 🥁
Much attention is paid to clean architecture and sustainable code evolution during the project, where testability is an impotant concern.

# ✨ Current features
* API endpoints for creating categories, instruments and shop items, as well as an EAV model for dynamic properties of instruments
* User authentication and authorization
* A standard account confirmation/recovery workflow with email via SendGrid
* Extensive unit tests for controller actions

# 💻 Tech stack
* **Backend**: ASP.NET Core 🖥
* **Database**: SQL Server ⛃ 

# 🗺 Roadmap
* Give stock managing functionalities to managers
* Implement pagination and sorting for API endpoints
* Implement external authentication logins
