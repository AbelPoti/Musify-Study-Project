# Musify Study project

This is a learning project that is actively in development. 📚

# Overview

This project aims to implement an ASP.NET Core Web API that should serve a webshop revolving around musical instruments. 🥁
Through the implementation, I strive to learn about state of the art REST practices, as well as to gain experience in writing clean and concise code that is both easily testable and scalable.

# ✨ Current features
* API endpoints for creating categories, instruments and shop items, as well as an EAV model for dynamic properties of instruments
* User authentication and authorization
* A standard account confirmation/recovery workflow with SendGrid

# 💻 Tech stack
* **Backend**: ASP.NET Core 🖥
* **Database**: SQL Server ⛃ 

# 🗺 Roadmap
* Document all methods with XML documentation tags
* Implement a robust unit testing suite for all actions
* Give stock managing functionalities to managers
* Implement pagination and sorting for API endpoints
* Implement external authentication logins
